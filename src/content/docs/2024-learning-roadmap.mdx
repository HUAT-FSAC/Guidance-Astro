### Ubuntu 安装方式对比

<Tabs>
  <TabItem label="🖥️ VMware 虚拟机" icon="seti:config">
    **推荐新手使用**
    
    通过 VMWare Workstation 部署 Ubuntu 虚拟机。
    
    **优点：**
    - 简单易上手
    - 与主系统隔离，不怕搞坏
    
    **安装步骤：**
    1. 下载 [VMware Workstation](https://www.vmware.com/go/getworkstation-win)
    2. 搜索 "vmware ubuntu 18.04 安装" 按教程操作
    3. 建议分配 4GB+ 内存，40GB+ 硬盘
    
    <Aside type="tip">
    VMWare 版本 16/17/18 都可以，Ubuntu 建议安装 18.04 LTS 版本以匹配 ROS Melodic。
    </Aside>
  </TabItem>
  
  <TabItem label="🪟 WSL2" icon="seti:windows">
    **适合 Windows 11 用户**
    
    Windows Subsystem for Linux，原生运行 Linux 环境。
    
    **优点：**
    - 无需大镜像文件
    - 启动快，安装/卸载方便
    - 与 Windows 融合度高
    
    **参考教程：** [Dev on Windows with WSL](https://dowww.spencerwoo.com/)
    
    <Aside type="caution">
    WSL 上可能存在图形界面画面撕裂问题（如 ros-rqt），请注意。
    </Aside>
    
    **更多参考：**
    - [知乎教程 1](https://zhuanlan.zhihu.com/p/377263437)
    - [少数派教程](https://sspai.com/post/74167)
    - [CSDN 教程](https://blog.csdn.net/qq401195092/article/details/133717025)
  </TabItem>
  
  <TabItem label="💿 物理机安装" icon="linux">
    **追求性能的选择**
    
    直接在真机上安装 Ubuntu 系统。
    
    **优点：**
    - 无虚拟化性能损耗
    - 完整的系统体验
    
    **缺点：**
    - 需要独立分区或硬盘
    - 操作有一定风险
    
    建议有经验的同学尝试双系统安装。
  </TabItem>
  
  <TabItem label="🐳 Docker" icon="seti:docker">
    **快速部署**
    
    使用 Docker 镜像部署 Ubuntu 环境。
    
    **优点：**
    - 即刻完成部署
    - 环境一致性好
    
    适合有 Docker 基础的同学。
  </TabItem>
</Tabs>

### 1.2 Linux 命令行操作

假定你已经成功完成了 Ubuntu 18.04 的安装，下面介绍基础操作。

<Card title="📚 推荐教程" icon="open-book">
  [Linux 漫游指南](https://libhitchhiker.eu.org/) - 一份优秀的 Linux 入门教程

#### 在哪里输入命令？

在 Ubuntu 桌面上：
- 按下 `Ctrl` + `Alt` + `T` 打开终端
- 或按 `Windows` 键，输入 `terminal` 后回车

<Aside type="tip">
如果使用 WSL，打开的命令窗口就是 Ubuntu 终端。
</Aside>

## 4. ROS 架构熟悉

### 4.1 ROS 介绍

<Card title="🤖 什么是 ROS？" icon="rocket">
  ROS 是一个适用于机器人的开源元操作系统。它提供：
  
  - 硬件抽象
  - 底层设备控制
  - 常用函数实现
  - 进程间消息传递
  - 包管理
  
  [详情参考 ROS 官方文档](https://wiki.ros.org/cn/ROS/Introduction)
</Card>
