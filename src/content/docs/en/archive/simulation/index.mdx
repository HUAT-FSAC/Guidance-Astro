## Overview

Simulation recreates the vehicle model and competition track in software so algorithms can run and be tested in a controlled environment.

<Aside type="tip" title="Why Simulation Matters?">
  - test algorithms without needing the real car
  - improve iteration speed and safety
  - expose potential issues earlier
  - use repeatable and controllable scenarios
</Aside>

## Quick Start

<Steps>

1. **Install dependencies**

   Make sure ROS Noetic and Gazebo are installed.

2. **Clone the repository**

   ```bash
   git clone https://github.com/HUAT-FSAC/fssim.git
   cd fssim
   ```

3. **Build the workspace**

   ```bash
   catkin_make
   source devel/setup.bash
   ```

4. **Launch the simulator**

   ```bash
   roslaunch fssim_gazebo fssim.launch
   ```

</Steps>

<Aside type="caution" title="Environment Requirement">
  FSSim currently expects Ubuntu 20.04 with ROS Noetic. A VM or WSL2 setup is usually the most practical starting point.
</Aside>
