## Frame design

### Structural concept

The car uses a **steel tubular space frame**:
- material: 4130 chromoly steel tube
- main tubes: Φ25.4 × 2.0 mm
- secondary tubes: Φ19.0 × 1.5 mm

### Design goals

1. **Meet rule requirements** - pass static safety inspection
2. **Optimize stiffness** - torsional stiffness above 2000 Nm/°
3. **Lightweighting** - frame mass below 35 kg

<Aside type="caution">
Frame design must comply strictly with the FSAC rulebook, especially the safety-structure requirements.
</Aside>

## Finite-element analysis

### Load cases

| Case | Description |
|------|----------|
| Torsion | diagonal-wheel lift with 1000 N |
| Bending | driver-compartment load |
| Frontal impact | 20G front impact |
| Side impact | 10G lateral impact |

### Software tools

- **ANSYS** - structural analysis
- **HyperMesh** - preprocessing
- **SolidWorks Simulation** - quick validation

## Aerodynamics

### Package configuration

- **Front wing** - generates downforce and improves front-tire grip
- **Rear wing** - primary downforce source
- **Diffuser** - uses underbody airflow for additional load

### Design priorities

- optimize airfoil shapes with CFD
- validate with wind-tunnel testing where possible
- balance downforce against drag
