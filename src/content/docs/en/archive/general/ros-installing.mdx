## Quick Install

<Steps>

1. Open a terminal

2. Run the one-click install script

   ```bash
   wget http://fishros.com/install -O fishros && . fishros
   ```

3. Follow the prompts and choose ROS Melodic

</Steps>

<Aside type="tip">
  The one-click script comes from [fishros](http://fishros.com/) and already configures domestic mirrors for faster installation.
</Aside>

## Manual Install

<Aside type="note">
  The steps below are provided for reference. The one-click method above is still the recommended path.
</Aside>

Before you start, make sure the Ubuntu basics are already configured: language, timezone, and `apt` sources.

<Steps>

1. **Configure the ROS repository**

   ```bash
   sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.tuna.tsinghua.edu.cn/ros/ubuntu/ `lsb_release -cs` main" > /etc/apt/sources.list.d/ros-latest.list'
   ```

2. **Add the PGP key**

   ```bash
   sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654
   ```

   <Aside type="tip">
     If that fails, try:
     ```bash
     curl -sSL 'http://keyserver.ubuntu.com/pks/lookup?op=get&search=0xC1CF6E31E6BADE8868B172B4F42ED6FBAB17C654' | sudo apt-key add -
     ```
   </Aside>

3. **Update package indexes**

   ```bash
   sudo apt update
   ```

4. **Install ROS Melodic**

   <Tabs>
     <TabItem label="Desktop Full (Recommended)">
       ```bash
       sudo apt install ros-melodic-desktop-full
       ```
       Includes ROS, rqt, rviz, common robotics libraries, 2D and 3D simulators, navigation, and perception packages.
     </TabItem>
     <TabItem label="Desktop">
       ```bash
       sudo apt install ros-melodic-desktop
       ```
       Includes ROS, rqt, rviz, and the common robotics libraries.
     </TabItem>
   </Tabs>

5. **Initialize rosdep**

   ```bash
   pip install rosdepc
   rosdepc init
   ```

6. **Set environment variables**

   ```bash
   echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc
   source ~/.bashrc
   ```

7. **Test the environment**

   Start the ROS core:

   ```bash
   roscore
   ```

   Open a new terminal and run:

   ```bash
   rosrun turtlesim turtlesim_node
   ```

   If a turtlesim window appears, open another terminal and run:

   ```bash
   rosrun turtlesim turtle_teleop_key
   ```

   If you can control the turtle with the arrow keys, the installation is working.

</Steps>
