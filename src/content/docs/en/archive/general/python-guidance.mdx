## Why Learn Python?

Even if most new members already spend more time with C or C++, Python still matters in the team's workflow.

<CardGrid stagger>
  <Card title="Faster Development" icon="rocket">
    ROS packages and scripts can often be prototyped much more quickly.
  </Card>
  <Card title="Easier Debugging" icon="magnifier">
    Dynamic typing and interactive tooling make quick experiments easier.
  </Card>
  <Card title="Rich Ecosystem" icon="puzzle">
    Libraries such as NumPy, OpenCV, and PyTorch are widely used in team projects.
  </Card>
</CardGrid>

<Aside type="tip">
  Becoming comfortable with Python will noticeably reduce friction in day-to-day development.
</Aside>

## Related Topics

### Python Virtual Environments

<Aside type="note">
  Virtual environments help avoid dependency conflicts and also let you manage multiple Python runtimes on the same machine.
</Aside>

<CardGrid>
  <Card title="Anaconda / Miniconda" icon="star">
    <Badge text="Recommended" variant="success" />

    Full-featured Python environment managers.
