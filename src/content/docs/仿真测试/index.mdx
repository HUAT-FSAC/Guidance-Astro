## 简介

{/* 视频文件待添加 */}

仿真是通过计算机模拟出**车辆模型**与**比赛赛道**，使各程序能够在模拟出的环境下运行并进行测试。

<Aside type="tip" title="为什么需要仿真？">
  - ✅ 无需实车即可测试算法
  - ✅ 提高测试效率与安全性
  - ✅ 提前发现潜在问题
  - ✅ 可重复、可控的测试环境
</Aside>

## 快速开始

<Steps>

1. **安装依赖环境**
   
   确保已安装 ROS Noetic 和 Gazebo

2. **克隆仿真仓库**
   
   ```bash
   git clone https://github.com/HUAT-FSAC/fssim.git
   cd fssim
   ```

3. **编译工作空间**
   
   ```bash
   catkin_make
   source devel/setup.bash
   ```

4. **启动仿真**
   
   ```bash
   roslaunch fssim_gazebo fssim.launch
   ```

</Steps>

<Aside type="caution" title="环境要求">
  FSSim 需要 Ubuntu 20.04 + ROS Noetic 环境，建议使用虚拟机或 WSL2。
</Aside>
