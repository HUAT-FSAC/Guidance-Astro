## 简介

定位与建图组通过使用**组合惯性导航单元**与多传感器（单目摄像头/激光雷达）配合进行：

<CardGrid stagger>
  <Card title="环境识别" icon="magnifier">
    识别周围赛道环境结构
  </Card>
  <Card title="车辆定位" icon="star">
    实时估计车辆位置与姿态
  </Card>
  <Card title="地图建立" icon="document">
    构建赛道全局地图
  </Card>
  <Card title="路径优化" icon="random">
    为规划控制提供基础
  </Card>
</CardGrid>

<Aside type="tip" title="重要性">
  精确的定位是自动驾驶的核心能力之一。没有准确的位置信息，车辆无法知道自己在哪里，也无法规划正确的行驶路径。
</Aside>

```
       └──────┬──────┘
              │
              ▼
┌─────────────────────────┐

└───────────┬─────────────┘
            │
            ▼
┌─────────────────────────┐
│    视觉/激光 SLAM       │ ←── 位姿校正 + 建图
└───────────┬─────────────┘
            │
            ▼
┌─────────────────────────┐
│      全局地图输出        │ ←── 供规划使用
└─────────────────────────┘
```
