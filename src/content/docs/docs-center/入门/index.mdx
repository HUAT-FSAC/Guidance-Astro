## 新成员 30 分钟快速上手

- [ ] 打开 /文档中心/ 与 /2025/ 入口，了解当前赛季结构
- [ ] 完成账号开通（邮箱、GitHub、云盘）
- [ ] 加入沟通群/频道并设置统一昵称
- [ ] 安装基础环境（见 /综合/）
- [ ] 确认部门方向与负责人

## 当天完成（Day 1）

- [ ] 阅读团队介绍、比赛背景、近两年成果
- [ ] 跑通一个最小任务或阅读一篇核心文档并做笔记
- [ ] 熟悉资源中心目录与命名规则（见 /文档中心/资源中心/）
- [ ] 在任务清单登记本周目标（见 /文档中心/流程与模板/）

## 第 1 周

- [ ] 完成环境配置并留存验证记录
- [ ] 参与一次评审或文档 Review
- [ ] 产出一次可归档成果（文档/脚本/图纸/会议纪要）
- [ ] 明确个人学习路径与阶段目标

## 第 1 月

- [ ] 完成一个小模块或子系统任务
- [ ] 提交一次验收并完成归档
- [ ] 参与一次复盘并输出改进项
- [ ] 能指导新人完成 30 分钟快速上手

## 账号与权限清单

| 项目 | 作用 | 申请方式/负责人 |
| --- | --- | --- |
| 学校邮箱 | 基础身份认证 | 由学院统一开通 |
| GitHub 组织 | 代码协作与 PR | 由负责人添加 |
| 云盘/共享盘 | 资料与大文件 | 由资料管理员开通 |
| 文档系统 | 文档编辑与发布 | 由负责人分配 |

## 角色入口与首个任务建议

### 算法部

- 入口：/2025/感知/ · /2025/定位建图/ · /2025/规控/ · /2025/仿真测试/
- 首周任务：复现一个基础流程或阅读一份模块设计文档
- 输出物：运行记录 + 问题清单 + 小结

### 电气部

- 入口：/2025/电气/
- 首周任务：熟悉电气架构与线束/原理图
- 输出物：架构笔记 + 疑点整理

### 机械部

- 入口：/2025/机械/
- 首周任务：阅读结构总览并打开一份 CAD
- 输出物：结构拆解笔记 + 风险点

### 项管部

- 入口：/2025/项管/
- 首周任务：熟悉对外资料与运营流程
- 输出物：资料清单 + 优化建议

## 工具与规范

- 命名规则：`YYYYMMDD_主题_负责人_vX`
- 归档位置：/文档中心/资源中心/
- 提交流程：/文档中心/流程与模板/

## 新人验收清单

- [ ] 能找到对应部门入口与负责人
- [ ] 能独立完成一次最小任务与归档
- [ ] 能提交任务清单与验收记录
- [ ] 能说明资源中心结构与命名规则

## 常见问题

- 不知道从哪里开始：先完成 30 分钟快速上手并找负责人确认方向
- 环境装不起来：在对应部门频道同步日志与截图
- 找不到资料：先查资源中心索引，再联系资料管理员
