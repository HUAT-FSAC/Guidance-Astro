## 使用建议

- 周会优先关注：`燃尽/关闭率`、`WIP 分布`、`CI 成功率`
- 里程碑评审优先关注：`Issue Lead Time`、`PR Cycle Time`
- 当 `CI 成功率` 连续下降时，优先排查最近 3 次失败构建链接

## 数据说明

| 指标 | 说明 | 数据来源 |
|------|------|----------|
| 总任务数 | 当前项目的所有 Issues 数量 | GitHub API |
| CI 通过率 | 最近 30 天内成功构建的比例 | GitHub Actions |
| 测试覆盖率 | 单元测试的代码覆盖比例 | Vitest |
| Bundle 体积 | 构建产物的总大小 | Astro 构建 |
| 里程碑进度 | 各里程碑的完成百分比 | 手动维护 |
