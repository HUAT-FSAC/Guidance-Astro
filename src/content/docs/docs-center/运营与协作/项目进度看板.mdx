## 使用建议

- 周会优先关注：`燃尽/关闭率`、`WIP 分布`、`CI 成功率`
- 里程碑评审优先关注：`Issue Lead Time`、`PR Cycle Time`
- 当 `CI 成功率` 连续下降时，优先排查最近 3 次失败构建链接
