# 团队运营

## 项目管理

### 进度跟踪

使用项目管理工具跟踪赛车设计制造进度：

| 工具 | 特点 | 推荐场景 |
|------|------|----------|
| 飞书项目 | 国产、免费 | 日常协作 |

### 里程碑规划

<Steps>
1. **设计阶段** - 9月-12月
2. **加工阶段** - 1月-4月
3. **装配调试** - 5月-6月
4. **测试优化** - 7月-8月
5. **参赛冲刺** - 9月-10月
</Steps>

## 文档管理

### 文档类型

1. **设计文档** - 技术方案、计算报告
2. **会议纪要** - 周会、评审记录
3. **培训资料** - 新人培训、技术分享
4. **赛事资料** - 规则、报名材料
5. **财务资料** - 预算、报销、账目

### 存储方案

<Aside type="tip">
推荐使用云盘 + 本地备份双保险：
- 飞书云文档（协作）
- 百度网盘/阿里云盘（备份）
- NAS（大文件存储）
</Aside>

## 赛事报名

### 报名材料

| 材料 | 截止时间 | 负责人 |
|------|----------|--------|
| 报名表 | 6月 | 运营 |
| 队员信息 | 6月 | 运营 |
| 技术报告 | 7月 | 各技术组 |
| 成本报告 | 7月 | 运营+财务 |
| 设计答辩 PPT | 8月 | 各技术组 |

### 注意事项

1. 密切关注官方通知
2. 提前准备材料模板
3. 预留修改时间
4. 多人交叉审核

## 团队建设

### 招新工作

<Steps>
1. 宣传造势 - 开学季海报、宣讲会
2. 简历收集 - 线上表单
3. 面试筛选 - 技术面 + 综合面
4. 入队培训 - 安全培训、技能培训
5. 试用考核 - 3个月考核期
</Steps>

### 团队活动

- **技术分享会** - 每两周一次
- **团建活动** - 每学期 1-2 次
- **总结大会** - 赛季结束
- **表彰大会** - 优秀队员表彰

## 财务管理

### 预算编制

| 类别 | 占比 | 说明 |
|------|------|------|
| 材料费 | 40% | 钢材、碳纤维等 |
| 外协加工 | 25% | CNC、3D 打印等 |
| 电子元器件 | 15% | 传感器、控制器 |
| 差旅费 | 10% | 参赛交通住宿 |
| 其他 | 10% | 工具、耗材等 |

### 报销流程

1. 填写报销单
2. 附发票/收据
3. 部长审批
4. 队长审批
5. 财务打款
