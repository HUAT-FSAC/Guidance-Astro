## 悬架系统概述

悬架系统是连接车轮与车身的重要部件，直接影响赛车的操控性能和稳定性。

### 悬架类型选择

我们采用**双叉臂独立悬架**，主要优势：

- 几何参数可调范围大
- 轮胎接地性好
- 适合高速过弯工况

### 关键参数

| 参数 | 前悬 | 后悬 |
|------|------|------|
| 轮距 | 1200mm | 1180mm |
| 主销内倾角 | 8° | - |
| 主销后倾角 | 4° | - |
| 外倾角 | -2° | -1.5° |
| 前束 | 0.5° | 0° |

## 转向系统

### 转向比设计

转向比的选择需要平衡：
- 转向灵敏度
- 驾驶员手力
- 高速稳定性

<Aside type="tip">
建议参考往届赛车数据和驾驶员反馈进行调整。
</Aside>

### Ackermann 转向几何

采用 **100% Ackermann** 几何，确保低速转弯时内外轮转向角差异合理。

## 减震器选型

### 推荐品牌

- **Öhlins TTX** - 高性能可调减震器
- **Bilstein** - 性价比之选
- **国产替代** - 成本控制时可考虑

### 调校要点

1. 压缩阻尼 - 控制过弯侧倾
2. 回弹阻尼 - 影响轮胎抓地力恢复
3. 预载 - 调整车身姿态
