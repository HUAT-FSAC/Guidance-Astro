## 为什么学 Python？

尽管大多数新队员已有 C/C++ 经验，但 Python 仍然在车队的开发工作中占据一席之地。

<CardGrid stagger>
  <Card title="✅ 开发更快" icon="rocket">
    Python 的 ROS 功能包开发周期更短
  </Card>
  <Card title="✅ 调试更容易" icon="magnifier">
    动态类型、交互式调试
  </Card>
  <Card title="✅ 生态丰富" icon="puzzle">
    NumPy、OpenCV、PyTorch 等库支持
  </Card>
</CardGrid>

<Aside type="tip">
  尽可能地掌握 Python 将**大大减轻**你在开发中的繁复工作。
</Aside>

## 相关推荐

### Python 虚拟环境

<Aside type="note">
  虚拟空间可以避免潜在的依赖冲突，同时也可以在同一系统环境下安装不同的 Python 运行时。
</Aside>

<CardGrid>
  <Card title="Anaconda / Miniconda" icon="star">
    <Badge text="推荐" variant="success" />
    
    功能完整的 Python 环境管理工具
    
    - [Anaconda 下载](https://repo.anaconda.com/archive/Anaconda3-2022.05-Linux-x86_64.sh)
    - [Miniconda 下载](https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh)
  </Card>
  <Card title="venv" icon="seti:python">
    <Badge text="轻量" variant="note" />
    
    Python 自带的虚拟环境模块，功能少但体积小
    
    [官方文档](https://docs.python.org/zh-cn/3/library/venv.html)
  </Card>
</CardGrid>
