# 🏎️ 我们的赛车

欢迎来到 HUAT FSAC 赛车展示页面。在这里，你可以了解我们历年来设计和制造的无人驾驶方程式赛车。

## 赛车设计理念

我们的赛车融合了前沿的工程技术与创新思维，每一代赛车都代表着团队对卓越的不懈追求：

- **轻量化设计**：采用碳纤维复合材料，实现最佳功率重量比
- **智能感知系统**：激光雷达、摄像头、IMU 多传感器融合
- **高效电驱系统**：自主研发电机控制器，实现精准扭矩输出
- **先进控制算法**：基于模型预测控制（MPC）的轨迹跟踪
