## 技术路线

### 感知系统

新一代感知系统将采用多传感器深度融合方案：

```
激光雷达 ─┬─→ 点云融合 ─→ 3D目标检测 ─┐
          │                           │
摄像头 ───┴─→ 图像融合 ─→ 语义分割 ───┴─→ 融合感知结果
```

**关键技术点：**
- 时间同步：硬件触发 + PTP 精确时钟同步
- 空间标定：自动化外参标定流程
- 融合算法：基于 Transformer 的多模态融合

### 定位建图

<Aside type="tip">
  我们将探索无 GNSS 环境下的纯视觉定位方案，以应对信号遮挡场景。
</Aside>

- 图优化 SLAM：基于因子图的多源信息融合
- 回环检测：结合视觉和几何特征
- 地图管理：增量式建图与地图更新

### 规划控制

**规划层：**
- 全局规划：基于赛道地图的最优路径生成
- 局部规划：实时避障与路径平滑

**控制层：**
- 纵向控制：速度规划 + PID/MPC 速度跟踪
- 横向控制：Stanley / Pure Pursuit / MPC
